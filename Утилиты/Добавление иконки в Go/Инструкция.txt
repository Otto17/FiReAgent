Добавление на Go иконки через утилиту "rsrc".
rsrc — это утилита для создания файлов ресурсов Windows (.syso), которые могут быть встроены в исполняемый файл Go.

1) Кидаем иконку в формате ico в папку.
2) Запускаем скрипт "Конвертировать ico в syso.bat".
3) Переносим созданный ".syso" файл в папку с исходным кодом программы на Go, затем обычным способом компилируем свою программу (например "go build ."), иконка сама автоматически подхватится и встроется в исполняемый файл при компиляции.
